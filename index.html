Privacy Policy for YouTube Speed Controller

Last updated: December 16, 2024

Data Collection

The YouTube Speed Controller extension is designed with privacy in mind. We collect and store:

  • Video playback speed preferences
  • UI opacity settings
  • Speed step value

Data Storage

All user preferences are stored locally on your device using Chrome's storage sync API. This allows your settings to sync across your Chrome browsers when you're signed in.

Data Usage

We do not:

  • Collect any personal information
  • Track your browsing history
  • Share any data with third parties
  • Send any data to external servers

Permissions

This extension requires the following permissions:

  • storage: To save your speed and UI preferences
  • activeTab: To interact with YouTube video player
  • tabs: To apply settings across YouTube tabs

Contact

If you have any questions about this privacy policy, please create an issue in our GitHub repository.
